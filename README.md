# BloodPressureMeasurementRecordMiniProgram
龙医护心-高血压记录小程序前端，采用Uniapp，vue编写。

## 基本信息

- [语言]：JS（Vue）
- [框架]：uniapp
- [核心库]：Uniapp、qiun-data-charts、UniUI、Uni-easyinput……
- [平台]：CentOS【腾讯云、阿里云的云服务】
- [测试]：go test(testing模块)
- [开发平台]：Windows10 采用Remote模式开发

## 主要业务

### A面

![A](.\docs\pic\A.png)

#### 血压录入

​	录入血压信息

#### 我的血压情况

​	显示我录入的血压信息情况

#### 我的治疗方案

​	浏览历史治疗方案、新增治疗方案

#### 我的检查报告

​	我的基本情况
​		链接B面-患者信息
​	我的检验指标
​		肌酐&BNP
​		心电
​		血压报告

#### 拍照上传

​	患者不懂自己的报告归类，直接上传交给后台人工处理
​		（二期）
​	利用人工智能CV，读取拍照的报告信息，自动生成字段填入患者的报告信息
​		（三期）

#### 舌苔脉象

​	利用深度学习神经卷积网络，处理大量的舌苔数据，根据患者传入的舌苔情况自动生成患者的状况信息。
​	根据患者的状况信息，在后台结合大数据的机器学习拟合归类，生成对目前患者的情况的一个改进建议
​	实现通过AI问诊舌苔

### B面

![B](.\docs\pic\B.png)

#### 我的账户

​	读取微信账户信息、地址、用户名、头像等信息

#### 患者信息

​	录入该账户对应的患者详细信息

#### 推送设置

​	对血压提醒推送进行的设置
